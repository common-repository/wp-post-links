Easy way to add unlimited permanent links after each of your posts.

== Description ==
With this plugin you can easily to add unlimited (can be configured how much) links after each post.
All you need is text file with links and descriptions hosted somewhere.

== Frequently Asked Questions ==

** What is the external text file structure?**
Each line of the external file should have two parts - link title and the link delimited by "|". For example:

url titlet text|http://url.com/
